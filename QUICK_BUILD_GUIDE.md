# Быстрая инструкция по сборке для Windows

## Самый простой способ

1. Откройте командную строку в папке проекта
2. Запустите:
   ```bash
   build_windows.bat
   ```
   или
   ```bash
   python build_all_windows.py
   ```

3. Дождитесь завершения сборки (5-15 минут)

4. Готовые приложения будут в папке `release/`

## Что будет собрано

- **WorkTimeTracker_Admin** - Админское приложение
- **WorkTimeTracker_User** - Пользовательское приложение  
- **WorkTimeTracker_Bot** - Telegram бот

## Распространение

Просто передайте папку или архив из `release/` администратору и пользователям.

Каждое приложение работает независимо - просто распакуйте папку и запустите .exe файл.

## Требования перед сборкой

- Python 3.10+
- Установлен PyInstaller: `pip install pyinstaller`
- Файл `secret_creds.zip` в корне проекта
- Файл `config.py` в корне проекта
